This repository is created for the flutter project we are working on. The title is Uniclass. This app allows students to see what the teachers have uploaded, commented on and the likes. The teachers can also communicate with the student through the comment section and can give assignments as well as provide learning materials. A notification page will notify the users of changes made. The features include: User CRUD operations, Notifications and Comments.
